CB-structure

CB-structure is a SaaS based crypto trading bot i've built on Flask and mySQL with integrated Binance API and Telegram Bot API for a startUp venture in Italy. User can subscribe to the bot from telegram and automate their crypto exchage on Binance based on signals from tradingview. The bot provides three different kind of trading strategies. The user can choose any strategy and the amount they want to invest. We receive the signals from tradingview through a webhook and execute orders on behalf of the users.
